# AICoach 📚

> 顶级AI学习私教 - 结合费曼学习法

本仓库包含 AICoach 技能的完整定义文件。

## 快速开始

进入 `AICoach/` 目录查看完整安装说明：

```bash
cd AICoach
```

## 目录结构

```
AICoach/
├── README.md       # 安装和使用说明
├── SKILL.md        # 技能定义文件
└── .claude/
    └── settings.json
```

---

## AICoach 功能预览

- 🎯 **个性化学习路径** - 根据你的水平和目标定制
- 🗺️ **知识树可视化** - 清晰的学习地图
- 📝 **自动资料管理** - 学习笔记、练习记录自动保存
- 💪 **费曼学习法** - 以教促学，真正掌握
- 🎉 **温柔鼓励风格** - 像一位耐心又风趣的导师
