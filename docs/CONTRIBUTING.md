# 贡献指南

感谢你愿意为 AICoach 贡献力量！🎉

---

## 如何贡献

### 1. 报告问题

- 在 [GitHub Issues](https://github.com/Cap-bit-mint/AICoach/issues) 创建新 Issue
- 选择合适的标签（bug / feature / question）
- 详细描述问题或建议

### 2. 提交修改

1. **Fork 本仓库**
2. **创建分支**：`git checkout -b feature/your-feature-name`
3. **提交更改**：`git commit -m 'Add: 描述你的更改'`
4. **推送到你的 Fork**：`git push origin feature/your-feature-name`
5. **创建 Pull Request**

### 3. 代码规范

- 使用中文编写注释和文档
- 保持代码风格一致
- 确保更改与 SKILL.md 的核心原则一致

---

## 技能优化方向

欢迎以下类型的贡献：

| 类型 | 描述 |
|------|------|
| 🔧 **功能增强** | 完善学习流程、诊断模板、练习设计 |
| 📚 **内容扩充** | 添加新的学习方法、示例、模板 |
| 🐛 **问题修复** | 修复触发词、响应逻辑、内容错误 |
| 📖 **文档完善** | 改进说明、添加使用案例 |

---

## 版本约定

- **MAJOR.MINOR.PATCH** (如 1.0.0)
  - MAJOR: 不兼容的重大更改
  - MINOR: 新功能（向后兼容）
  - PATCH: 问题修复

---

## 行为准则

- 保持友善和尊重
- 接受建设性的反馈
- 专注于为用户创造价值
